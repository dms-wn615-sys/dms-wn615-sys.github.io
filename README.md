# 오리 블록 게임

8×8 보드에 오리 모양 블록을 배치해 줄·열을 없애는 퍼즐 게임입니다. 7개국어·BGM·점수 순위 지원.

## 실행 방법

### PC / Android
1. `index.html` 파일을 브라우저에서 엽니다.
2. **게임 시작** 버튼을 클릭하거나 Enter/Space 키를 누릅니다.

### iPhone
- Safari에서 `index.html` 열기 → 하단 터치 버튼으로 플레이
- **공유 → 홈 화면에 추가**로 앱처럼 설치 가능
- iOS 앱 빌드: [`IOS.md`](IOS.md) 참고 (Mac + Xcode 필요)

### Android 앱
- [`PLAYSTORE.md`](PLAYSTORE.md) 참고

## 조작법

| PC 키보드 | 모바일 터치 |
|---|---|
| ← → | ◀ ▶ 이동 |
| ↓ | ▼ 빠르게 내리기 |
| ↑ | ↻ 회전 |
| Space | ⏬ 즉시 낙하 |
| P | 일시정지 버튼 |

## 규칙

- 떨어지는 오리 블록을 맞춰 가로 한 줄을 채우면 해당 줄이 사라집니다.
- 줄을 없앨수록 점수가 올라가고, 10줄마다 레벨이 올라가 속도가 빨라집니다.
- 블록이 더 이상 들어갈 공간이 없으면 게임 오버입니다.
