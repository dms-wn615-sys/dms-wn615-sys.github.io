# 정부24 — 게임제작업 등록 가이드 (타일 매치)

「게임산업진흥에 관한 법률」에 따라 **게임을 제작·유통하기 전** 관할 시·군·구에 **게임제작업 등록**이 필요합니다.  
(배급만 하는 경우는 **게임배급업** — 타일 매치는 **제작업**만 해당)

## 신청 링크

| 항목 | URL |
|------|-----|
| **정부24 민원** | https://www.gov.kr/mw/AA020InfoCappView.do?CappBizCD=13700000177 |
| **정부24 홈** | https://www.gov.kr → 검색: `게임제작업 등록` |

신청 방법: **인터넷(정부24)** · 방문 · 우편  
처리 기간: **약 3일** (관할 구청)  
신청 자격: **누구나** (개인 개발자 포함 — 관할 구청 확인 권장)

---

## 1. 타일 매치 해당 여부

| 항목 | 내용 |
|------|------|
| **등록 종류** | **게임제작업** (배급업 아님) |
| **게임** | 타일 매치 (Tile Match) |
| **개발자** | 배서현 (DMS Games) |
| **유통** | 토스콘솔(앱인토스) 예정 |

영업 **개시 전** 등록하는 것이 안전합니다. 미등록 제작 시 법상 제재 대상이 될 수 있습니다.

---

## 2. 구비 서류

### 본인이 제출 (게임제작업)

| 서류 | 타일 매치 |
|------|-----------|
| **게임제작업 등록신청서** | 정부24 작성 (별지 4호) |
| **제작시설 및 장비 명세서** | `store-assets/gov24/facility-equipment-list.txt` → 주소·장비 수정 후 PDF |
| **임대차계약서 사본** | **임차한 경우만** (자가 주택이면 보통 불필요 — 구청 확인) |
| **사업계획서** | **배급업만** — 제작업 해당 없음 |

### 공무원 확인 (해당 시)

- 법인등기부등본 (법인)
- 건물등기부등본 (임차 + 소유자≠임대인인 경우)

---

## 3. 신청서 작성 (복사용)

상세: **`store-assets/gov24/application-fields.txt`**

| 항목 | 입력 예 |
|------|---------|
| 대표자 | 배서현 |
| 이메일 | dms-wn615@hanmail.net |
| 등록 구분 | ☑ 게임제작업 |
| 제작품목 | 타일 매치 — 모바일 퍼즐 (앱인토스) |

---

## 4. 수수료 (관할마다 상이 — 사전 확인)

| 항목 | 참고 금액 |
|------|-----------|
| 접수 수수료 | 약 20,000~25,000원 |
| 등록면허세 | 약 40,500원(서울 기준 등) — **매년** 납부 |

---

## 5. 절차

1. **관할 구청** 확인 (주민등록상 주소지 구·군·시)
2. 정부24 로그인 → **게임제작(배급)업 등록** → **게임제작업** 선택
3. 신청서 작성 + **제작시설·장비 명세서** 첨부
4. 수수료 납부
5. 처리 완료 문자/알림 → **게임제작업 등록증** 수령 (방문 또는 우편)

---

## 6. 등록 후 의무

- **대표자·영업소·제작품목 변경** → 변경등록 (7일 이내 등, 구청 안내 따름)
- **등록면허세** 매년 납부
- **영업 폐지** → 7일 이내 등록증 반납

---

## 7. GRAC · 토스와 순서

권장 순서:

```
1. 게임제작업 등록 (정부24)     ← 지금
2. GRAC 등급분류 (간소화)       → docs/GRAC.md
3. 토스 콘솔 앱 등록·출시       → docs/TOSS-CONSOLE-INPUT.md
```

GRAC·토스 신청 시 **제작업 등록증** 번호·사본을 요구할 수 있으므로 등록증을 보관하세요.
