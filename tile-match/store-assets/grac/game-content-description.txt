2. 게임 개요
--------------------------------------------------------------------------------
  겹쳐진 타일을 탭해 하단 슬롯으로 옮기고, 같은 타일 3개를 모으면
  제거되는 3매치 퍼즐 게임입니다. 모든 타일을 제거하면 레벨 클리어,
  슬롯(7칸)이 가득 차면 게임 오버입니다.

3. 게임 진행 방법
--------------------------------------------------------------------------------
  (1) 홈 화면에서 [계속하기] 또는 레벨 선택
  (2) 보드에서 선택 가능한(위에 가리지 않은) 타일 탭
  (3) 슬롯에 같은 타일 3개가 모이면 자동 제거
  (4) 모든 타일 제거 시 레벨 클리어 (별·코인 보상)
  (5) 부스터: 실행 취소, 섞기, 힌트, 슬롯+1 (게임 내 코인으로 구매)
  (6) 클럽 채팅·일일 토너먼트·리더보드 (로컬 시뮬레이션, 자체 서버 없음)

4. 주요 기능
--------------------------------------------------------------------------------
  - 20,000개 레벨 (1,000장 × 20레벨, 절차적 생성)
  - 12개 여행 테마 배경
  - 7개국어 (한국어, English, 日本語, 中文, Español, Français, Deutsch)
  - 배경음악·효과음 (설정 on/off, Web Audio 자체 생성)
  - 오프라인 플레이 (PWA)
  - Google 로그인 (선택, 진행 데이터 기기 저장)
  - 광고: 현재 버전 없음 · 추후 Google AdMob 배너 광고 도입 예정
