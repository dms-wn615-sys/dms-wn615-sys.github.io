토스 콘솔 — 플레이 화면 (등급분류 게임 = 출시 게임, 화면 동일)
================================================================================

토스 안내: 「등급분류를 받은 게임과 출시하려는 게임의 화면이 동일해야 해요」

타일 매치: GRAC 심의·앱인토스(.ait) = 동일 게임 빌드
→ 아래 2장을 **같은 이미지**로 양쪽 란에 각각 업로드

■ ① 자체등급분류 / GRAC 게임물 화면 (2장)
  store-assets/toss/rating-screens/self-rating/
    self-rating-01-gameplay-start.png
    self-rating-02-gameplay-match.png

■ ② 앱인토스 플레이 화면 (2장)
  store-assets/toss/rating-screens/ait-play/
    ait-play-01-gameplay-start.png
    ait-play-02-gameplay-match.png

※ ①과 ②는 **쌍별로 픽셀 동일** (같은 세션 캡처 후 복사)
※ 편집·합성·자르기 금지

■ 생성
  npm run toss:rating-screens

■ GRAC 증빙이 「게임물 등급분류증명서」만인 경우
  자체등급분류(스토어) 란 생략 가능 — 앱인토스 2장만 제출
